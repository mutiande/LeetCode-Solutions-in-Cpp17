* 动态规划，爬到`n`阶之前必然先爬到`n - 1`或`n - 2`阶，因此爬到`n`阶的方法数就是爬到`n - 1`阶和`n - 2`阶的方法数之和
* 爬到1阶有1种方法，爬到2阶有2种方法，以此作为初始解即可求出后续解

```cpp
class Solution {
public:
    int climbStairs(int n) {
        if (n <= 2) return n; 
        vector<int> dp(n + 1);
        dp[1] = 1;
        dp[2] = 2;
        for (int i = 3; i <= n; ++i)
        {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        return dp[n];
    }
};
```
